How to Set Up


Setting up this system is easy as it seems...

First, you need to attach the Sensors/Actuators you that want to use to the Verdin Device.

Second, you need to open the Verdin App on your smartphone/tablet/PC, search for the Verdin Device connected via wireless, and then select which types of Sensors/Actuators you have attached.

Finally, the system will start to catch a lot of information from your garden. Keep your eyes on the Verdin App for a moment...
